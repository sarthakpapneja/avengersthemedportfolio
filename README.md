# S.H.I.E.L.D. Portfolio | Sarthak Initiative

Built with React, Vite, and tactical CSS.

## 🚀 Live Mission
**Deployment URL**: [https://avengersthemedportfolio.vercel.app/](https://avengersthemedportfolio.vercel.app/)

## 🛠 Tech Stack
- **Core**: React 19
- **Build Tool**: Vite 6
- **Aesthetic**: Futuristic HUD Design (Vanilla CSS)
- **Deployment**: Vercel

## 📂 Project Structure
- `src/App.tsx`: The primary mission component containing the HUD interface.
- `src/main.tsx`: Entry point for the S.H.I.E.L.D. system.
- `src/index.css`: Global tactical styling.

## ⚙️ Development
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the HUD system:
   ```bash
   npm run dev
   ```
